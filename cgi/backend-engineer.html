Modules

The CGI training will be delivered via live instructor-led sessions, pre-recorded lectures, hands-on labs, and a variety of assessments. The curriculum for this track is subdivided into 5 parts and you will take all the modules in them.

Part I. Internet & Web Fundamentals

  • 1. Introduction to Internet
  • 2. Web Fundamentals
  • 3. Basic HTML and CSS

II. Python Programming Essentials

  • 1. Introduction to Python & Installation
  • 2. Python Programming Basics
  • 3. Logical Execution & Control Flow
  • 4. Collection Data Structures
  • 5. Advanced Data Structures in Python

III. Design and Development I

  • 1. Setting up JavaScript Dev Environment
  • 2. Introduction to JavaScript Syntax
  • 3. Introduction to Operating Systems
  • 4. Introduction to Git and GitHub
  • 5. Data Basics of Programming Servers with JavaScript
  • 6. Backend Development with Node/Express.js
  • 7. Introduction to APIs
  • 8. Authentication and Authorization Methodologies

IV. Design and Development II

  • 1. Introduction to Django & Setting Up
  • 2. Introduction to MVC, MVVM
  • 3. Designing URL & Views
  • 4. Templating & Static Files
  • 5. Django Database Schemas
  • 6. Models in Django
  • 7. User Testing Techniques

V. Design and Development III

  • 1. Design Principles and TDD with JavaScript
  • 2. Building Interactive Web Experiences
  • 3. SQL for Backend Development
  • 4. Database Design and Integration
  • 5. Software Principles – YAGNI, KISS, DRY
  • 6. Object-Oriented Design and SOLID
  • 7. Software Design Patterns
  • 8. Caching and CDNs
  • 9. Web Sockets

VI. Configuration and Deployment

  • 1. Introduction to Deployment Techniques
  • 2. Configuring APIs with Node, Express, and PostgreSQL
  • 3. Configuring Django Backend Servers
  • 4. Deploying Backend Servers
  • 5. Final Portfolio Project
