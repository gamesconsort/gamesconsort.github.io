Modules

The CGA internship curriculum has been carefully designed to help you significantly grow your knowledge base in data science over 6 months. The training will be delivered via instructor-led live sessions, pre-recorded lectures, hands-on labs, and a variety of assessments.

Part I: Foundations

  • 1. Introduction to Data and Analytics
  • 2. The Data Value Chain: Decisions, Operations, Monetisation
  • 3. Data Strategy Design and Execution
  • 4. Business Thinking for Data Scientists
  • 5. Systems Thinking for Data Scientists
  • 6. Success Factors for Data Science
  • 7. Data Science Project Workflow
  • 8. Careers in Data Science
  • 9. Bulding Strong Data Science Portfolios

II: Python and Math Fundamentals

  • 1. Python Programming Basics
  • 2. Python Stack for Data Science
  • 3. Single and Multivariable Calculus
  • 4. Fundamentals of Linear Algebra
  • 5. Statistics and Probability
  • 6. Introduction to Information Theory
  • 7. Logic, Algorithms and Graphs

III: Working with Data

  • 1. Data Structures and Algorithms
  • 2. Cloud Data Architectures
  • 3. Databases and SQL Fundamentals
  • 4. Introduction to Descriptive Analytics
  • 5. Data Acquisition and Manipulation
  • 6. Data Prep with Paxata on DataRobot
  • 7. EDA and Statistical Thinking
  • 8. Probability Distributions and Statistical Inference
  • 9. Decision Making Under Uncertainty
  • 10. Business Data Anaysis & EDA in SQL
  • 11. Storytelling: Communicating with Data
  • 12. Data Visualisation with Power BI

IV: Machine Learning

  • 1. Machine Learning: Pattern Discovery
  • 2. Machine Learning: Predictive Analytics
  • 3. Feature Engineering Best Practices
  • 4. Ensembles and Model Combination
  • 5. Evaluating Machine Learning Models
  • 6. Statistical Testing & ML Experiment Design
  • 7. Hands-on AutoML on DataRobot
  • 8. Scalable Analytics and ML on Big Data
  • 9. ML with Probabilistic Graphical Models
  • 10. Statistical Data Bias Detection
